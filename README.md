# Jyväskylän Teekkariyhdistyksen Website

This repository contains the source code for the Jyväskylän Teekkariyhdistys (Jyty) website.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installing

1. Clone the repository:
```sh
git clone https://github.com/yourusername/jyty-web.git
```
2. Navigate to the project directory
```sh
 cd jyty-web
```
3. Install the dependencies
```sh
 npm install
```

### Running the Development Server

To start the development server, run
```sh
npm run dev
```
The website will be available at http://localhost:3000
